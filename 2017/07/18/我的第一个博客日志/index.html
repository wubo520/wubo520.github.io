创建gitbug账号
输入
用户名

密码 字母+数字 八位以上
配置邮箱
点击头像
选择settings
点击Emails
登录主邮箱 点击激活
添加次邮箱
Add email address
登录次邮箱 点击激活

    生成秘钥
        点击头像
            选择settings
        点击 SSH and GPG keys

        打开 Git Bash
            输入
                ssh-keygen -t rsa -C"主邮箱"
            一路回车
                秘钥生成完毕

            用户目录下有一个.ssh文件夹
                目录下有俩文件
                    id_rsa
                    id_rsa.pub  √
                    不能使用编辑器打开，只能使用记事本

                    把内容都复制
                    把复制的内容粘贴到key中
                    点击add
    配置
        打开Git Bash命令行
        查看配置
            git config -l
        需要配置name和email
        配置name
            git config --global user.name "名字"
        配置email
            git config --global user.email "主邮箱"
        验证是否成功
            git config -l
            找到name和email
本地项目传到github上
    点击头像边上的加号
    选择 New Repository

    填写项目名
    填写描述

    选择Public

    点击 Create repository

    创建文件
        README

    需要输入命令
        打开Git bash命令
        输入提示的命令
        项目上传

    需要上传
        git add 文件名
        git add .       所有文件
        git commit -m"url"
        git push
 最终版上传

echo # MyProject >> README.md

git init

git add README.md

git commit -m “first commit”

git remote add origin https://github.com/springleft/douban

git pull origin master（如果出错：git pull –rebase origin master）

git push -u origin master
